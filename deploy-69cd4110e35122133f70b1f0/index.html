Accessible sur Minecraft Java & Bedrock

Rejoindre TerraVerse sur Minecraft Java Edition

Windows, MacOS & Linux

#1

Lancer Minecraft Java Edition puis appuyer sur le bouton Multijoueur.

#2

Cliquer sur Ajouter un serveur et saisir ce que tu veux dans le champ "Nom du serveur".

#3

Dans "Adresse du serveur", rentrer l'IP play.terraverse.fr et cliquer sur Rejoindre.

Rejoindre TerraVerse sur Minecraft Bedrock

Mobile / Pocket Edition

#1

Lancer Minecraft: Bedrock Edition et appuyer sur le bouton Jouer.

#2

Aller dans l'onglet Serveurs, puis défiler vers le bas et appuyer sur Ajouter un serveur.

#3

Saisir TerraVerse dans "Nom du serveur" et play.terraverse.fr dans "Adresse du serveur" avec le port 19132, puis cliquer sur Jouer.

Rejoindre TerraVerse sur Minecraft Console

Xbox One / Xbox Series / PlayStation

#1

Télécharger l'application BedrockTogether sur App Store ou Google Play.

#2

Ouvrir l'application, entrer play.terraverse.fr et le port 19132, puis lancer la connexion. Vérifier que le téléphone et la console sont sur le même Wi-Fi.

#3

Lancer Minecraft, aller dans Amis, puis cliquer sur BedrockTogether pour rejoindre TerraVerse.

Rejoindre TerraVerse sur Minecraft Switch

Nintendo Switch

#1

Aller dans Paramètres console > Internet > Paramètres Internet. Sélectionner le Wi-Fi, puis Modifier les paramètres. Mettre DNS manuel : 104.238.130.180 et 8.8.8.8.

#2

Lancer Minecraft avec un compte Nintendo Switch Online, aller dans Serveurs, rejoindre un serveur en vedette puis ouvrir le menu d'ajout de serveur personnalisé.

#3

Entrer play.terraverse.fr et le port 19132, cocher l'ajout à la liste, puis sélectionner TerraVerse pour rejoindre.

!

Cette méthode Switch peut changer selon les mises à jour Nintendo/Mojang. À vérifier régulièrement.
